SUMMARY

The first chapter introduced four types of javascript values: booleans, strings, numbers and undefined values.
These values can be created by typing in their names, for example (null, true) or value, (12, "abc").
I also learnt about operators and logic.

Programs are built with statements, statements contain expressions.
Using statements one after the other allows the program to be executed from top to bottom,
disturbances can be brought into the flow using conditional and looping statements like
(if, else, switch) and (while, do, for).
Bindings are used to file pieces of data under a name and are useful for tracking state in programs.
Functions are special values that contain a piece of program.
You can call a function by writing functionName(argument).
